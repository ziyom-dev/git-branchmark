Тема урока продолжение

  • Создание ветки - git branch namebranch
  • слияние веток - git merge название ветки
  • переход на коммит и ветку - git checkout
  • Удаление веток - git branch -d (имя ветки)
  • Создание и переход на ветку - git checkout -b (имя ветки)
  • Клонирование гит репозитория- git clone (ссылка на репозиторий)
  • удаление коммита - git reset (название коммита)
  • История коммитов - git log --oneline (название коммита)
